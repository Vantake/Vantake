# Contributing

This repository is under active development.

## Local setup
1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env.local` and fill required values
3. Run: `npm run dev` 

## Quality checks
- Lint: `npm run lint`
- Typecheck: `npm run typecheck`
- Build: `npm run build`

## Commit style
Prefer small, focused commits with clear messages (e.g. `feat: ...`, `fix: ...`, `chore: ...`).
